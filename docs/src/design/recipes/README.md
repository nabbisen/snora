# Recipe index

Recipes are copy-paste doc patterns built from Snora Design primitives.
They are not stable public APIs — they are starting points you adapt for
your application.

See [Recipes and dogfood process](../../contributing/recipes.md) for the
nine-section format, promotion criteria, and the dogfood feedback template.
